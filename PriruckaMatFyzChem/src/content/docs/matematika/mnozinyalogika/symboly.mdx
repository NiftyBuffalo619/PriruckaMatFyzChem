# Symboly Množin

| Symbol | Význam | Příklad |
| --- | --- | --- |
| <Latex formula="\in" /> | Je prvkem (elementem) množiny. | <Latex formula="1 \in \reals" /> Jedna je elementem(součástí) realných čísel. |
| <Latex formula="\notin" /> | Není prvkem (součástí) množiny. | <Latex formula="\pi \notin \Z" />  Pí není součástí celých čísel. |
| <Latex formula="\subseteq , \subseteqq" /> | Podmnožina(Neostrá inkluze množin) | <Latex formula="A \subseteq B" /> A je podmnožinou B |
| <Latex formula="\subset" /> | Vlastní Podmnožina(Ostrá inkluze) | <Latex formula="A \subset B" /> A je vlatní podmnožinou B |
| <Latex formula="\supseteq , \supseteqq" /> | Nadmnožina | <Latex formula="A \supseteq B" /> A je  nadmnoužinou B |
| <Latex formula="\supset" /> | Vlastní Nadmnožina | <Latex formula="A \supset B" /> A je vlastní nadmnoužinou B |
| <Latex formula="\cup" /> | Sjednocení množin | <Latex formula="A \cup B" /> A sjednocení B |
| <Latex formula="\cap" /> | Průnik množin | <Latex formula="A \cap B" /> A průnik B |
