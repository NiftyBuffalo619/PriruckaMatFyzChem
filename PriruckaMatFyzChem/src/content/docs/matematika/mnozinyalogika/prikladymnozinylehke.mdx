## 1. Příklad
 <Latex formula="A = \{ x \in \natnums; x \leq 6 \}" />
 <details>
    <summary>Zobrazit Řešení</summary>
    <Latex formula="A = \{ 1;2;3;4;5;6 \}" />
 </details>
## 2. Příklad
 <Latex formula="B = \{x \in \mathbb{Q}; 3x = 5 \}" />
 <details>
    <summary>Zobrazit Řešení</summary>
    <Latex formula="B = \{ \frac 3 5 \}" />
 </details>
## 3. Příklad
 <Latex formula="C = \{x \in \Z; 3x = 5 \}" />
 <details>
    <summary>Zobrazit Řešení</summary>
    <Latex formula="3x = 5, x \in \Z" />
    <Latex formula="x \notin \Z" />
    **x** Nemá v oboru celých čísel žádné řešení, proto množina C je množina prázdná
    <Latex formula="C = \{ \varnothing \}" />
 </details>
## 4. Příklad
 <Latex formula="D = \{x \in \Z; -4 \le x \lt 4 \}" />
 <details>
   <summary>Zobrazit Řešení</summary>
   <Latex formula="D = \{ -4; -3; -2; -1; 0; 1; 2; 3 \}" />
 </details>
## 5. Příklad
<Latex formula="E = \{x \in \Z; -2 \lt x \le 2\}" />
<details>
   <summary>Zobrazit Řešení</summary>
   <Latex formula="E = \{ -1;0;1;2 \}" />
</details>
## 6. Příklad
Jsou dané dvě množiny **A** , **B**. Zjistěte jejich průnik a sjednocení.
<Latex formula="A = \{ 1 ; 2 ; 3; 4 \}" />
<Latex formula="B = \{ 1; 3; 5 \}" />
<details>
   <summary>Zobrazit Řešení</summary>
   <Latex formula="A \cup B = \{ 1; 2; 3; 4; 5 \}" />
   <Latex formula="A \cap B = \{ 1; 3 \}" />
</details>
