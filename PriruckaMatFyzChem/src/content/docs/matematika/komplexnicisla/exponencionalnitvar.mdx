Komplexní číslo lze zapsat i tzv. Exponencionálním tvarem. Z Eulerovy Identity plyne
<Latex formula="|z|( \cos \alpha + i \sin \alpha ) = |z| e^{i \alpha}" />

### Převod z algebraického do exponencionálního tvaru
Vyjadřete číslo z v exponencionálním tvaru
<Latex formula="z = 1 - \sqrt{3} i " />

Nejprve najdeme jeho absolutní hodnotu 
<Latex formula="|z| = \sqrt{1^2 + (- \sqrt{3})^2} = \sqrt{1 + 3} = 2" />
Určíme argument komplexního čísla 
<Latex formula="\cos \alpha = \frac{a}{|z|} = \frac{1}{2}" />
<Latex formula="\alpha \in \lbrace \frac{\pi}{3} + 2k \pi ; \frac{5 \pi}{3} + 2k \pi  \rbrace" />
<Latex formula="\sin \alpha = \frac{b}{|z|} = \frac{- \sqrt{3}}{2} = \frac{- \sqrt{3}}{2}" />
<Latex formula="\alpha \in \lbrace \frac{4 \pi}{3} + 2k \pi ; \frac{5 \pi}{3} + 2k \pi  \rbrace" />
Jejich společný argument alpha, absolutní hodnotu vememe a dosadíme 
<Latex formula="z = |z|e^{i \alpha} = 2e^{i \frac{5 \pi}{3}}" />

### Převod z exponencionálního tvaru do algebraického tvaru
Vememe minulej příklad a převedeme ho do algebraického tvaru 
<Latex formula="2e^{i \frac{5 \pi}{3}}" />
Nejprve vypočítáme obě hodnoty sinu a cosinu 
<Latex formula="\cos \frac{5 \pi}{3} =  \frac{1}{2}" />
<Latex formula="\sin \frac{5 \pi}{3} = - \frac{\sqrt{3}}{2}" />
Nyní jenom dosadíme pro **a** a **b** 
<Latex formula="\cos \alpha = \frac{a}{|z|} \implies a = |z| \cos \alpha = 2 \frac{1}{2} = 1" />
<Latex formula="\sin \alpha = \frac{b}{|z|} \implies b = |z| \sin \alpha = 2 -\frac{\sqrt{3}}{2} = - \sqrt{3}" />
Tyto hodnoty dosadíme (Nezapomeňte na imaginární hodnotu)
<Latex formula="z = 1 - \sqrt{3} i " />
A vidíme , že nám vyšel algebraický tvar
